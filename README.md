# Picture Sorter
<h3>Sort, tag, and explore your images - faster, smarter, and visually!</h3>

Easily organize your images in any folder (and optionally its subfolders) with powerful sorting algorithms!

- [x] Sort images using popular algorithms: Quicksort, Mergesort, Heapsort, and more
- [x] Choose from multiple sorting criteria: name, date, size, brightness, Color Variance and more
- [x] Auto generate custome tags with AI
- [x] Optionally include subfolders for full-folder organization
- [x] Lightweight and easy to use

Note: This will not change the sorting in your Explorer. Its just gives you a visual Reprensentation (with the Filename).
